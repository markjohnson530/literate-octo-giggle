Commands:

    General:


  • /sell - show the cobblestone sell gui
  • /vote - list vote sites
  • /hat - put a held item on your head
  • /discord - get our Discord invite in chat

    You can create a private teleportation point using server homes. Each player can set up to three.


  • /home <name> - teleport to your home point
  • /sethome <name> - create a home point
  • /delhome <name> - delete a home point

    Commands regarding the Stone Miner Scoreboard:


  • /togglescoreboard or /tsb - show/hide the scoreboard
  • /myscore- show your score
  • /showscoreboard - show the current scoreboard listings in chat
  • /stoneminers - list these commands in-game

    Teleportation Commands:


  • /warp <name> - teleport to a specific warp
  • /warps - list all warps
  • /spawn - teleport to spawn
  • /tpa <player> - send a teleportation request to a player
  • /tpaccept <player> - accept a teleportation request from a player.
  • /tpdeny <player> - deny a teleportation request from a player.

    Chat:


  • /msg <player> - send a private message to an online player
  • /r - privately reply to your last messaged player
  • /mail read - read your mail
  • /mail send <player> - send mail to a player. They do not have to be online.
  • /mail clear - clear your mail

    Money:


  • /pay <player> <amount> - send someone money
  • /balance - view your balance
  • /baltop - view a list of the richest players
