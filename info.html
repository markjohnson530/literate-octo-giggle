World Modifications:

  • The world border is set to 1 million in each direction from 0,0. This is significantly smaller than the default 30 million, but big enough to allow for impressive builds to be created that can realistically remain undiscovered for years.
  • Phantoms do not naturally spawn but spawn eggs can be purchased with in-game money at the shop.

Dragon Egg Event:

  • Every Saturday at 7pm EST, a Dragon Egg is generated in a random location within 20,000 blocks of spawn. The coordinates are publicly released in chat and the discord.

Currency:

    In-game money can be obtained on the server by selling cobblestone using /sell. You cannot sell any other item.
    Any use of the terms “money,” “$,” or equivalent within this server refers exclusively to an in-game, fictional currency used for entertainment and gameplay purposes. This virtual currency is not real money, carries no cash value, and cannot be exchanged, converted, redeemed, or traded for real-world currency, goods, or services.

Stone Miner Scoreboard:

    The Stone Miner scoreboard is a leaderboard seperated from money featuring the top ten players that have mined the most stone. Commands regarding this can be found in the Command section.
